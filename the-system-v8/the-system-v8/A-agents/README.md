This folder contains your AI agent definitions.

## What's Here

```
A-agents/
├── adam-agent.md         → Your COO - manages the team, workflows, and memory
├── copywriter-agent.md   → Writes all your content in your voice
├── gatekeeper-agent.md   → Reviews quality before publishing
├── researcher-agent.md   → Finds angles, data, and supporting material
├── strategist-agent.md   → Analyzes from business/growth perspective
├── analyst-agent.md      → Turns raw material into structured documents
├── devils-advocate-agent.md → Challenges assumptions and finds blind spots
├── chief-of-staff-agent.md  → Synthesizes multiple perspectives into briefs
├── tom-agent.md          → Your guide to the ABC-TOM system
└── README.md             → You are here
```

## How Agents Work Together

Adam (your COO) orchestrates the team. You talk to Adam, Adam manages everyone else.

### Content Track
```
YOU → Adam → Researcher → Copywriter → Gatekeeper → Adam → YOU
```

### Sales Track
```
YOU → Adam → Strategist → Copywriter → Gatekeeper → Adam → YOU
```

### Thinking Track (Decision Brief)
```
YOU → Adam → Strategist → Devil's Advocate → Chief of Staff → Gatekeeper → Adam → YOU
```

### Thinking Track (Summary)
```
YOU → Adam → Analyst → Copywriter → Gatekeeper → Adam → YOU
```

You are always the CEO. Adam runs the team. Agents execute. You decide.

## Adding New Agents

To add a new agent:

1. Create `[role]-agent.md` in this folder
2. Define their:
   - Core identity
   - Responsibilities
   - Required reading (what files they need)
   - Output format
   - Quality standards
3. Update the team hierarchy if needed

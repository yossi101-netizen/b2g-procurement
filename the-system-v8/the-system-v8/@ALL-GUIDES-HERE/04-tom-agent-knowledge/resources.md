| Tool | What It Does | Cost | Why Tom Uses It |
|------|--------------|------|-----------------|

## Quick Recommendations

**"What tools do I need?"**
> Start with Claude (you have it) and Obsidian (free). That's the core. Everything else is optional.

**"How do I generate images?"**
> Replicate. Pay per image, no subscription. Use the Nano Banana model for illustrations.

**"Should I pay for Claude Max?"**
> Pro ($20) is enough to start. Max ($100) if you use it heavily and want unlimited.

**"What about ChatGPT?"**
> I use Claude. It thinks differently, more like a collaborator. But the ABC-TOM system can work with almost any AI.

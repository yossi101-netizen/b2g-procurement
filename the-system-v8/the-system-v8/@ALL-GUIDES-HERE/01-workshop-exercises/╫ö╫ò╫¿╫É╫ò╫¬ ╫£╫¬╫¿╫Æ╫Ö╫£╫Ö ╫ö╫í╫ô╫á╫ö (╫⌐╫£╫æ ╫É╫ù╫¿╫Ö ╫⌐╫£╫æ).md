### הכנה: התקנת קבצי הפרויקט

1. הורידו את קובץ ה-ZIP ופתחו אותו.
2. גררו את התיקייה שנפתחה לתוך ה-Obsidian Vault שלכם.
3. ב-Windows? לפעמים נוצרת תיקיית ".obsidian" מיותרת בתוך ה-Vault. התעלמו ממנה.
4. ודאו שאתם רואים את כל התיקיות (A-agents, B-brain, C-core, T-tools וכו') בסרגל הצד ב-Obsidian. הן בתוך תיקיית the-system שחילצתם מה-ZIP.

### הכנה: פתיחת Claude Code

4. פתחו שיחה חדשה ב-Claude Code.
5. **חשוב:** בחרו את תיקיית the-system כתיקיית העבודה. ככה Claude יודע איפה לקרוא, לערוך וליצור קבצים.

### שלב 1: בחרו את המסלול שלכם

לפני שמתחילים, בחרו את המסלול שמתאים לעבודה שלכם:

**תוכן** (Content)
פוסטים, מאמרים לבלוג, ניוזלטרים, יצירת תוכן.
*בשבילכם אם: אתם משווקים, מעצבים, יוצרי תוכן, כל מי שמפרסם תוכן. גם מתאים למנהלים שכותבים thought leadership או עדכונים לצוות.*

**מכירות** (Sales)
הצעות ללקוחות, הצעות מחיר, מיילי מכירה, תקשורת עסקית.
*בשבילכם אם: אתם יזמים, פרילנסרים, יועצים, כל מי שכותב ללקוחות.*

**חשיבה** (Thinking)
תדריכי החלטות, סיכומי ישיבות, סיכומי ראיונות משתמשים, מסמכי מוצר, עדכוני סטייקהולדרים.
*בשבילכם אם: אתם מנהלי מוצר, מנכ"לים, בעלי עסקים, מנהלים, מנהלי צוותים, כל מי שמארגן מידע ומקבל החלטות. גם מתאים לעדכוני דירקטוריון, סקירות רבעוניות, ומסמכי אסטרטגיה.*

**לא בטוחים מה לבחור?** שאלו את עצמכם:
- אתם מפרסמים דברים שאנשים קוראים? → **תוכן**
- אתם כותבים ללקוחות ומוכרים? → **מכירות**
- אתם מקבלים החלטות ומסכמים מידע? → **חשיבה**

### שלב 2: מלאו את המידע שלכם

פתחו את הקובץ `B-brain/01-about-me.md`

מלאו את הטקסט עם המידע שלכם (העתיקו-הדביקו מהמסמך שמילאתם מראש).

**אין לכם "עסק"?** אין בעיה. זה עובד לכולם:
- אם אתם עובדים בחברה: כתבו על התפקיד שלכם, הצוות שלכם, והאנשים שאתם משרתים
- אם אתם פרילנסרים: כתבו על מה שאתם עושים ומי הלקוחות שלכם
- אם אתם חוקרים/לומדים: כתבו על מה שאתם רוצים ליצור ובשביל מי

**אל תשנו שמות קבצים.** הפרומפטים מקושרים אליהם.

פתחו את תיקיית הדוגמאות שמתאימה למסלול שלכם:

**טיפ:** ככל שהדוגמאות שלכם טובות יותר, ככה הפלט טוב יותר. בחרו דברים שאתם גאים בהם.

## שלב ב׳: הריצו את הצוות החדש שלכם

*עכשיו עברו ל-Claude Code. הריצו פרומפט אחד בכל פעם. חכו שיסיים לפני שמדביקים את הבא.*

כל 4 הפרומפטים נמצאים בתיקיית המסלול שלכם: `T-tools/02-prompts/[המסלול שבחרתם]/`

**פרומפט 1. ספרו לי על עצמכם**

- העתיקו את כל התוכן של `01-learn-about-me.md`
- הדביקו ב-Claude Code ושלחו.
- Claude קורא את התשובות שלכם ומעדכן אוטומטית את קבצי ה-C-core: `project-brief`, `icp-profile`, `voice-dna`.

**פרומפט 2. לימוד סגנון הכתיבה שלי**

- העתיקו את `02-learn-my-style.md` והדביקו.
- Claude קורא את 3 הדוגמאות ומעדכן את קבצי ה-Core, האייג'נטים, וה-Memory.

**פרומפט 3. יצירת הפלט שלכם**

- העתיקו את קובץ ה-`03-...` מתיקיית המסלול שלכם והדביקו.
- מלאו את הפרטים (הנושא, הלקוח, או החומר הגולמי בהתאם למסלול).
- אדם (סמנכ"ל התפעול שלכם) מנהל את כל הצוות כדי ליצור את הפלט.

מה כל מסלול מייצר:

- **תוכן** בחרו: פוסט, מאמר לבלוג, או ניוזלטר. 3 סוכנים עובדים יחד: חוקר מוצא זוויות → קופירייטר כותב בקול שלכם → גייטקיפר בודק איכות.
- **מכירות** הצעה מותאמת ללקוח. 3 סוכנים עובדים יחד: אסטרטג מנתח את המצב → קופירייטר כותב בקול שלכם → גייטקיפר בודק איכות.
- **חשיבה** בחרו: תדריך החלטות או סיכום מסמך.
  - תדריך החלטות: 4 סוכנים מתווכחים על ההחלטה שלכם (אסטרטג → עורך דין של השטן → ראש מטה → גייטקיפר)
  - סיכום: 3 סוכנים מעבדים את החומר הגולמי שלכם (אנליסט → קופירייטר → גייטקיפר)

**פרומפט 4. סגירת ה-Loop**

- העתיקו את `04-close-the-loop.md` והדביקו.
- אדם בודק מה יצרנו, מתעד מה עבד ומה לא ב-M-memory, ומקדם תובנות חזקות ל-C-core.
- זה מה שגורם למערכת להשתפר עם הזמן.

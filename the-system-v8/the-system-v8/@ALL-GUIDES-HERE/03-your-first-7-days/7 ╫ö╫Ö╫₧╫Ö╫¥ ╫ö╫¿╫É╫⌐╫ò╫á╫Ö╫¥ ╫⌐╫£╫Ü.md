# 7 הימים הראשונים אחרי הסדנה

בניתם צוות AI תוך כמה שעות. עכשיו מה?

הנה תוכנית פשוטה. דבר אחד ביום. בלי לחץ. תדלגו על יום אם רוצים. המערכת לא הולכת לשום מקום.

---

## יום 1: תיצרו משהו אמיתי

עשיתם את התוצר הראשון בסדנה. עכשיו תעשו שוב, אבל עם נושא אמיתי שלכם.

**מה לעשות:**
1. פתחו את תיקיית המסלול שלכם ב-`T-tools/02-prompts/` (content, sales, או thinking)
2. העתיקו את פרומפט `03-...` והדביקו ב-Claude Code
3. הפעם בחרו נושא אמיתי. משהו שאתם באמת צריכים לכתוב/להחליט/לסכם.
4. כשזה מוכן, קראו את התוצר ב-`O-output/`. קרוב לקול שלכם? אם לא, זה נורמלי. זה משתפר.
5. הריצו את `04-close-the-loop.md` כדי לתעד מה למדתם. זה מה שעושה את התוצר הבא טוב יותר.

**בונוס:** נסו מסלול אחר. אם עשיתם content בסדנה, נסו sales או thinking. אותה מערכת, תוצר שונה לגמרי.

---

## יום 2: תאכילו את המוח

ככל שהסוכנים יודעים יותר, ככה הם מבצעים טוב יותר. היום הוא על להפוך אותם לחכמים יותר.

**מה לעשות:**
1. פתחו את תיקיית הדוגמאות ב-`B-brain/02-my-samples/` (01-content, 02-sales, או 03-thinking)
2. הוסיפו 2-3 דוגמאות נוספות של עבודה אמיתית שלכם. הדברים הכי טובים שכתבתם.
3. הריצו שוב את פרומפט `02-learn-my-style.md` מתיקיית המסלול שלכם
4. השוו את התוצר החדש לאתמול. תראו את ההבדל.

**למה זה חשוב:** 3 דוגמאות זה המינימום. 5-7 זה איפה שזה מתחיל להיות טוב באמת. הסוכנים לומדים את הקצב שלכם, את בחירת המילים, אפילו את אורך הפסקאות.

**טיפ:** יש לכם הערות, רעיונות, או קבצים שאתם לא בטוחים מה לעשות איתם? תזרקו אותם ל-`B-brain/04-INBOX/`. זו המגירה. תסדרו אחר כך, או תבקשו מ-Claude לסדר בשבילכם.

---

## יום 3: הכירו את פרומפטים הבונוס

יש 9 פרומפטים בונוס ב-`T-tools/02-prompts/BONUS/`. היום תחקרו אותם.

**התחילו עם אחד מאלה:**

| פרומפט | מה הוא עושה | טוב בשביל |
|---|---|---|
| `09-content-calendar.md` | מתכנן את התוכן שלכם ל-2 השבועות הקרובים | אם אתם רוצים מבנה ועקביות |
| `10-repurpose-content.md` | לוקח פוסט אחד והופך אותו ל-5 פורמטים | אם יש לכם פוסט טוב שיכול להיות יותר |
| `12-create-hook-bank.md` | בונה אוסף של משפטי פתיחה | אם אתם מסתכלים על דף ריק לעתים קרובות |

**איך:** פתחו את הקובץ, העתיקו את הפרומפט, מלאו את הפרטים, הדביקו ב-Claude Code.

אל תנסו את כל ה-9 היום. בחרו אחד. תשחקו איתו.

## יום 4: צרו את הסוכן הראשון

בצוות שלכם כבר יש את אדם (סמנכ"ל תפעול), קופירייטר, גייטקיפר, חוקר, אסטרטג, ועוד. אבל תמיד יש מקום למישהו חדש.

**מה לעשות:**
1. פתחו את `T-tools/02-prompts/BONUS/05-create-new-agent.md`
2. חשבו: איזו משימה חוזרת אתם צריכים עזרה בה?
3. מלאו את הפרטים והדביקו את הפרומפט

**רעיונות לסוכנים:**
- **עורך** בודק את הכתיבה שלכם לבהירות וזרימה לפני שהגייטקיפר בודק
- **חוקר** אוסף מידע ונתונים לפני שאתם כותבים
- **מתרגם** מתאים את התוכן שלכם בין עברית לאנגלית
- **מנהל רשתות חברתיות** ממחזר פוסט אחד לכמה פלטפורמות
- **מנהל תקשורת לקוחות** כותב טיוטות לכל ההודעות ללקוחות בטון שלכם

**זכרו:** תפקיד אחד ברור לכל סוכן. "עושה הכל" זה סוכן גרוע. "מומחה בשיווק" זה סוכן טוב.

---

## יום 5: בנו את הסקיל הראשון

סקילים הם מדריכים מקצועיים. הם מלמדים את הסוכנים איך לעשות משימות ספציפיות.

**מה לעשות:**
1. פתחו את `T-tools/02-prompts/BONUS/06-create-new-skill.md`
2. חשבו: איזה סוג תוכן אתם יוצרים באופן קבוע שיש לו כללים ספציפיים?
3. מלאו את הפרטים והדביקו את הפרומפט

**שימו לב:** המערכת שלכם כבר כוללת סקילים חזקים: פוסט חברתי, בלוג פוסט, ניוזלטר, הצעות מכירה, סיכום מסמכים, סיכום פגישות, PRD, עדכון סטייקהולדרים, כתיבה בעברית, ועוד. בדקו את `T-tools/01-skills/` כדי לראות את כולם.

**רעיונות לסקילים חדשים:**
- **סקיל תיאור מוצר** אם אתם כותבים תיאורי מוצרים או שירותים
- **סקיל case study** אם אתם יוצרים סיפורי הצלחה של לקוחות
- **סקיל pitch deck** אם אתם יוצרים מצגות למשקיעים או ללקוחות
- **סקיל code review** אם אתם בודקים קוד באופן קבוע

**התוצאה:** קובץ מדריך ב-`T-tools/01-skills/` שכל סוכן יכול להשתמש בו כשהוא עושה את סוג העבודה הזה.

**מדריך מהיר ל-T-tools:**
- **סקילים (Skills)** = ידע. מה הסוכן יודע איך לעשות. (למשל: "איך לכתוב ניוזלטר")
- **פרומפטים (Prompts)** = טריגרים. מה אתם מדביקים כדי להתחיל משהו. (למשל: "תכתוב פוסט על X")
- **וורקפלואוס (Workflows)** = צנרת. תהליכים רב-שלביים שמחברים סוכנים וסקילים. (למשל: "טיוטה ← ביקורת ← תיקון ← אישור")

כלל אצבע: אם זה ידע, תעשו סקיל. אם זה קיצור דרך שאתם מדביקים, תעשו פרומפט. אם יש 3+ שלבים עם העברות בין סוכנים, תעשו וורקפלואו.

## יום 6: סגירת ה-Loop (גרסה מעמיקה)

עד עכשיו יצרתם תוכן, הוספתם דוגמאות, אולי בניתם סוכן חדש. הגיע הזמן לסגירת Loop מעמיקה.

**מה לעשות:**

1. **עדכנו את הזיכרון.** פתחו את `M-memory/learning-log.md` והוסיפו מה למדתם השבוע:
   - איזה תוצר נשמע כמוכם?
   - מה לא עבד?
   - איזה פידבק הייתם נותנים לסוכנים?

2. **חדדו את הקול.** פתחו את `C-core/voice-dna.md`. ספציפי מספיק? הוסיפו דוגמאות קונקרטיות:
   - מילים שאתם תמיד משתמשים בהן
   - מילים שאתם אף פעם לא משתמשים בהן
   - כמה ארוכות הפסקאות שלכם
   - איך אתם פותחים טקסט

3. **סגרו את ה-Loop.** הריצו פרומפט `04-close-the-loop.md` מתיקיית המסלול שלכם. זה מקדם את התובנות הכי טובות מ-Memory חזרה ל-Core, כדי שבפעם הבאה הסוכנים יתחילו מבסיס חזק יותר.

4. **סדרו את ה-INBOX.** בדקו את `B-brain/04-INBOX/`. יש שם משהו? תעבירו לתיקייה הנכונה, או תבקשו מ-Claude לסדר בשבילכם.

## יום 7: הפעלת שיפור אוטומטי ותכנון קדימה

שני דברים היום. אחד עושה את המערכת חכמה יותר, ואחד עושה אתכם מאורגנים יותר.

**בוקר: הפעילו את לולאת השיפור האוטומטית**
1. פתחו את `T-tools/02-prompts/BONUS/07-enable-auto-revision-loop.md`
2. העתיקו והדביקו את הפרומפט
3. מעכשיו, הקופירייטר והגייטקיפר עובדים ביחד אוטומטית. אתם רואים רק את הגרסה המאושרת.

**אחה"צ: תכננו את השבוע הבא**
1. פתחו את `T-tools/02-prompts/BONUS/09-content-calendar.md`
2. מלאו את הנושאים, התדירות ותקופת הזמן
3. תקבלו לוח תוכן מובנה שאפשר לעקוב אחריו

## אחרי יום 7: תמשיכו

המערכת מצטברת. שבוע 1 מחוספס. שבוע 10 קסם.

**הרגל שבועי (5 דקות):**
- הריצו את הסקירה השבועית: `T-tools/02-prompts/BONUS/13-weekly-review.md`
- עדכנו את `M-memory/learning-log.md` עם מה שעבד

**הרגל חודשי (15 דקות):**
- הוסיפו דוגמאות כתיבה חדשות ל-`B-brain/`
- בדקו ועדכנו את `C-core/voice-dna.md`
- הריצו את ה-Loop: `04-close-the-loop.md` מתיקיית המסלול שלכם
- סדרו את `B-brain/04-INBOX/`
- צרו סוכן או סקיל חדש אם צריך

**כשסיימתם פרויקט ב-O-output/:**
- הוסיפו `-done` לשם התיקייה (למשל: `01-linkedin-post/` הופך ל-`01-linkedin-post-done/`)
- ככה העבודה הפעילה נשארת נקייה וקלה לסריקה

**כשאתם תקועים:**
- כתבו `/tom` ושאלו את השאלה
- בדקו את `@ALL-GUIDES-HERE/04-tom-agent-knowledge/common-questions.md`
- שאלו בקבוצת הוואטסאפ של בוגרי הסדנאות

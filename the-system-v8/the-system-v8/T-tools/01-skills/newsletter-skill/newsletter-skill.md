# Newsletter Skill

Write newsletters people look forward to. Not ones they archive.

## When to Use

- Writing email newsletters (Substack, Mailchimp, ConvertKit, etc.)
- Creating recurring content for subscribers
- Converting ideas into newsletter format

---

## Newsletter Fundamentals

**What makes newsletters work:**
- A subject line that earns the open
- A personal voice (not a corporate broadcast)
- One main idea per issue (depth beats breadth)
- Value that justifies the inbox space

**Ideal length:**
- **Quick hit:** 300-500 words (one insight, one takeaway)
- **Standard:** 500-1,000 words (one topic explored well)
- **Deep dive:** 1,000-2,000 words (thorough, for engaged audiences)

**Frequency rule:** Consistent beats frequent. Weekly > sporadic daily.

---

## The Subject Line

Open rate lives or dies here. You have ~40 characters before mobile cuts you off.

### Subject Line Formulas

**Curiosity gap:**
```
The mistake I keep making
What nobody tells you about [topic]
I was wrong about [thing]
```

**Specific value:**
```
3 tools that saved me 10 hours this week
The email template with a 40% reply rate
How I [specific result] in [timeframe]
```

**Personal/Story:**
```
A weird thing happened last Tuesday
The conversation that changed my mind
I almost didn't send this
```

**Direct/Simple:**
```
[Topic]: what works and what doesn't
My honest take on [trending topic]
Quick update + something useful
```

### Subject Line Rules
1. Under 50 characters (under 40 is better)
2. Lowercase often outperforms Title Case (feels personal)
3. No clickbait — deliver on the promise
4. Preview text (first line) extends the subject line — use it

## Newsletter Structures

### Structure 1: One Big Idea
```
[Subject line]

[Personal hook: A story, observation, or moment]

[The idea: What you realized/learned/discovered]

[Why it matters: Connect to the reader's world]

[How to apply it: 2-3 practical steps]

[Closing: Personal sign-off, question, or preview of next issue]
```

### Structure 2: Curated + Commentary
```
[Subject line]

[Quick personal intro: 2-3 sentences]

[Item 1: Link + your take (not just a summary)]
[Item 2: Link + your take]
[Item 3: Link + your take]

[One insight that connects them all]

[Closing]
```

### Structure 3: Behind the Scenes
```
[Subject line]

[What happened this week/month]

[What worked (with specifics)]

[What didn't work (with honesty)]

[What I'm trying next]

[What you can learn from this]

[Closing]
```

### Structure 4: The Tutorial Newsletter
```
[Subject line: The outcome]

[Hook: Why this matters right now]

[Step 1] — with example
[Step 2] — with example
[Step 3] — with example

[Results you can expect]

[Closing + CTA]
```

## Writing Rules for Newsletters

### Rule 1: Write Like You're Emailing One Person
```
Corporate: "Dear valued subscribers, we're excited to share..."
Newsletter: "Hey — something interesting happened this week."
```

### Rule 2: The First Line Is Everything
After the subject line, the first sentence decides if they keep reading. Start with:
- A story moment
- A surprising fact
- A question that resonates
- A confession

Never start with: "This week I want to talk about..." or "Happy [day of week]!"

### Rule 3: One Main Idea Per Issue
Don't try to cover everything. Go deep on one thing. Subscribers remember depth, not breadth.

### Rule 4: Make It Scannable
- Short paragraphs (1-3 sentences)
- Subheadings for longer issues
- Bold for key points
- Bullet points for lists
- White space is your friend

### Rule 5: End With Connection
```
Bad ending: "Thanks for reading!"
Good ending: "Hit reply and tell me — what would you do differently?"
Good ending: "Next week: the part nobody talks about."
Good ending: "If this was useful, forward it to someone who needs it."
```

## Newsletter-Specific Tactics

### Building the Habit
- Send on the same day/time consistently
- Use a recognizable sender name
- Create a format readers can expect
- Reference previous issues to build continuity

### Growing the List
- Include a share/forward CTA in every issue
- Create a "best of" issue for new subscribers
- Cross-promote on social (tease, don't duplicate)
- The content itself is the best growth tool

### Measuring What Works
- **Open rate:** Subject line quality
- **Click rate:** Content relevance + CTA clarity
- **Reply rate:** Personal connection + engaging questions
- **Unsubscribe rate:** Expectation mismatch or frequency issues

---

## What to Avoid

1. **"Dear subscribers"** — You're not writing a formal letter
2. **Multiple CTAs** — One ask per email
3. **Apologizing for gaps** — Just deliver value now
4. **Wall of links** — Commentary > curation
5. **Corporate tone** — Newsletter = personal channel
6. **Burying the value** — Lead with the good stuff

---

## Quality Checklist

Before sending:

- [ ] Subject line would make YOU open this?
- [ ] First sentence earns the scroll?
- [ ] One clear main idea?
- [ ] Scannable formatting (short paragraphs, bold, bullets)?
- [ ] Personal voice (not corporate)?
- [ ] Specific examples or stories?
- [ ] Clear ending with one CTA?
- [ ] Read it as if you're the subscriber — would you finish it?
- [ ] Preview text (first line) works with the subject line?

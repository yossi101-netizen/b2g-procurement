# Prompt: Create a New Skill

Use this to add a new skill to your agents' toolkit.

Copy and paste this into your chat (fill in your skill idea first):

---

```
I want to create a new skill for my agents. Here's what I need:

---
Skill name: [e.g., "newsletter-skill", "email-sequence-skill", "twitter-thread-skill"]
What it teaches: [What should agents learn from this skill?]
When to use: [In what situations should agents use this skill?]
---

Please do the following:

1. First, study the existing skills to understand the structure and style:
   - Read T-tools/01-skills/social-post-skill/social-post-skill.md
   - Read T-tools/01-skills/professional-email-skill/professional-email-skill.md
   - Read T-tools/01-skills/research-briefing-skill/research-briefing-skill.md
   - Notice the format: frontmatter, when to use, basics, structures, rules, templates, checklists

2. Read the brand core so the skill aligns with the brand:
   - C-core/project-brief.md
   - C-core/voice-dna.md
   - C-core/icp-profile.md

3. Create a new folder in T-tools/01-skills/ with the skill name (e.g., T-tools/01-skills/newsletter-skill/)

4. Create the skill file inside that folder (e.g., newsletter-skill.md):
   - Use the same structure as existing skills
   - Include frontmatter (name, description)
   - Add "When to Use" section
   - Explain the basics of this content type
   - Provide proven structures and templates
   - Include writing rules specific to this format
   - Add quality checklist
   - Include examples of good and bad practices
   - End with the signature: > **© Tom Even**
> Workshops & future dates: [www.getagents.today](https://www.getagents.today)
> Newsletter: [www.agentsandme.com](https://www.agentsandme.com)

5. Update the relevant agent files in A-agents/:
   - Think about which agents would use this skill (copywriter, researcher, analyst, strategist, etc.)
   - For each relevant agent, add the new skill to their "Required Reading" or "Skills" section
   - Add any specific instructions for using this skill in that agent's context
   - Not every skill is for every agent. A writing skill is for copywriter. A research skill is for researcher. A formatting skill might be for all of them.

6. Update M-memory/learning-log.md:
   - Add an entry documenting the new skill
   - Note when agents should use it

7. Consider if we need a new workflow:
   - Should we create a workflow file in T-tools/03-workflows/ for this content type?
   - If yes, create it following the structure of content-workflow.md

Show me the new skill file and confirm all updates are done.
```

# Prompt: Create Content

Choose your output type, fill in your idea, then copy and paste into your chat:

---

```
I want to create content. Here's my idea:

---
[PASTE YOUR IDEA HERE - can be a rough thought, a topic, a story, or just a few words]
---

Output format (pick one):
- Social media post (LinkedIn / Facebook / Instagram / Twitter/X)
- Blog post / article
- Newsletter issue

Adam, please manage this project:

1. Pre-flight check: Read C-core/project-brief.md, C-core/voice-dna.md, and C-core/icp-profile.md. If any of these files are empty or still have placeholder text, STOP and tell me which files need to be filled first.

2. Read the workflow in T-tools/03-workflows/content-workflow.md

3. Read my content samples: B-brain/02-my-samples/01-content/

4. Read the right skill from T-tools/01-skills/:
   - Social post → social-post-skill
   - Blog post → blog-post-skill
   - Newsletter → newsletter-skill

5. Read past learnings: M-memory/learning-log.md

6. Delegate to the Researcher (read A-agents/researcher-agent.md first):
   - Find interesting angles on this topic
   - Look for trends, data points, or comparable content
   - Surface insights that would make this piece stand out
   - Create a new folder in O-output/ with the next available number, and a _process/ subfolder inside it
   - Save as _process/researcher-brief.md

7. Review the research, then delegate to the Copywriter (read A-agents/copywriter-agent.md first):
   - Write a first draft using the Researcher's insights
   - Match MY voice and style from the samples
   - Apply the right skill format for the chosen output type
   - Save as _process/copywriter-draft.md with notes about choices made

8. Send to the Gatekeeper (read A-agents/gatekeeper-agent.md first):
   - Review against all standards (voice, quality, format)
   - Save as _process/gatekeeper-review.md
   - If approved, create final-[type].md at the PROJECT FOLDER ROOT (not inside _process/) — ready to publish
   - If not approved, manage revisions

9. Update M-memory/learning-log.md with what you learned

Show me the final result and tell me if it's ready to publish or needs revision.
```

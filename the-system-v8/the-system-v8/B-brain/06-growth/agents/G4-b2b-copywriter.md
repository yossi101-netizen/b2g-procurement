## Voice Rules (non-negotiable)

See Voice DNA v2 for full specification. Summary:
- British English. Always.
- 8-word maximum for emphasis sentences
- 30-word maximum for any sentence
- Problem before solution. Every time.
- Named mechanisms. Never paraphrase.
- Numbers. Not adjectives.
- Anti-pitch close. Every article.

## Supplier Naming Rule (PERMANENT NON-NEGOTIABLE — CEO ORDER 2026-04-22)

KritiKaal operates a "Certified Bench, Disclosed on Qualification" model. This rule is absolute and applies to all published content, all GEO assets, and all external communications:

**NEVER name specific manufacturing partners, tanneries, or facilities by name in any published content.**

Use cluster + certification language only:

✅ CORRECT: "LWG Gold certified facilities in the Calcutta Leather Complex"
✅ CORRECT: "SA8000-certified factories in the Chennai corridor"
✅ CORRECT: "Our Tier 1 manufacturing bench includes multiple LWG Gold certified facilities"
❌ FORBIDDEN: Any specific facility name (e.g. "Dugros Leather", "Gateway Leathers", or any named factory)

**Why this rule exists:** KritiKaal's supply chain identity is protected commercial information. Specific partner names are disclosed only at the qualification stage and in production contracts — never in public content. Violating this rule exposes KritiKaal's bench to competitive approach and undermines the managed manufacturing model.

G6 is authorised to veto any content that names specific manufacturing partners. This is an unconditional veto.
